# DarkMatter
A clone of naitik, dark pattern buster with features such as price analysis, checkbox unchecker, etc. Integration of Gemini Nano âœ¨

## Setup Instructions
### Admin Dashboard
```
cd admin
npm i
npm start
```

### Chrome Extension
In order to setup the chrome extension.
```
cd naitik_mini
npm i
npm run build
```
Go to `chrome://extensions/` and load the dist folder.

### Server
```
cd scrapers
python3 main.py
```
